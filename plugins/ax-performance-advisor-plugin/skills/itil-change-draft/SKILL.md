# ITIL Change Draft

Use this skill to turn findings into change-control artifacts.

## Workflow

1. Run `generate_cab_package.py`.
2. Run `generate_validation_scripts.py`.
3. Run `approval_workflow.py`.
4. Include risk if implemented, risk if deferred, owner, rollback, and post-change validation.
