# Performance Debt Governance

Use this skill to manage performance work as a backlog.

## Workflow

1. Run `update_performance_debt_register.py`.
2. Run `calculate_debt_interest.py`.
3. Run `estimate_cost_of_delay.py`.
4. Run `continuous_improvement_scorecard.py`.
5. Prioritize by severity, recurrence, owner, cost, and validation readiness.
