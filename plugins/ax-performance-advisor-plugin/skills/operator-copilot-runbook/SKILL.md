# Operator Copilot Runbook

Use this skill to support interactive operations and incident review.

## Workflow

1. Run `operator_copilot_context.py`.
2. Run `generate_runbook.py`.
3. Run `incident_replay.py`.
4. Run `root_cause_narrative.py`.
5. Present evidence, why it matters, next check, owner, and validation.
