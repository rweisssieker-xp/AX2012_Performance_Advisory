# Approval Workflow Governance

Use this skill when findings need formal lifecycle management.

## Workflow

1. Run `scripts/approval_workflow.py`.
2. Use `scripts/generate_cab_package.py` and `scripts/generate_validation_scripts.py`.
3. Update state only after owner review and evidence-backed validation.
