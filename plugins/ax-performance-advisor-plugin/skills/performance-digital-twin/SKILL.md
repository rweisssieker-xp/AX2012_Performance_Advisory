# Performance Digital Twin

Use this skill when modeling an AX environment for scenario analysis.

## Workflow

1. Run `scripts/build_digital_twin.py`.
2. Review AOS nodes, databases, batch jobs, query families, hot tables, and calendar.
3. Use the twin as input for blast-radius, portfolio, archive, and topology analysis.
