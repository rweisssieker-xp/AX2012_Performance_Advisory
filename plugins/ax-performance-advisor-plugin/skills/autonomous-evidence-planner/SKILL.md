# Autonomous Evidence Planner

Use this skill when confidence is low or evidence is incomplete.

## Workflow

1. Run `scripts/plan_next_evidence.py`.
2. Run `scripts/confidence_drilldown.py`.
3. Collect only the missing read-only sources needed to increase confidence.
