# Natural Language Root Cause Chat

Use `scripts/ai_insights.py` with `--question` to produce an evidence-grounded answer. Always cite finding IDs, likely causes, and missing evidence gaps. Treat the answer as an advisory diagnosis, not an automatic remediation command.
