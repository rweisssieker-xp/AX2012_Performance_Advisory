# Custom Code Risk Map

Use this skill for customizations and vendor code.

## Workflow

1. Run `customization_risk_map.py`.
2. Run `custom_code_ownership_graph.py`.
3. Run `link_query_to_xpp.py` when Trace Parser/DynamicsPerf evidence exists.
4. Route findings by owner, class, table, report, service, and query family.
