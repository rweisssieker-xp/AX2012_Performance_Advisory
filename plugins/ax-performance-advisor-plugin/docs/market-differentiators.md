# Market Differentiator USPs

This pack adds product-positioning and management differentiation artifacts.

## Features

- Vendor-neutral comparison against generic database observability.
- Migration readiness signals.
- Resilience score.
- Knowledge graph export of findings, playbooks, modules, and tables.
- Process owner scorecards.
- Evidence marketplace for missing high-value evidence.
- Value realization opportunities.

## Run

```powershell
python .\scripts\market_differentiators.py `
  --evidence .\evidence\IT-TEST-ERP4CU `
  --output .\out\market-differentiators.json
```

The dashboard embeds these results in the `More USPs` tab.
