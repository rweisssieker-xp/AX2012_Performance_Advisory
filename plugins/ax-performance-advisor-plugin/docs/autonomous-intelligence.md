# Autonomous Intelligence

This pack adds AI/USP artifacts for investigation planning and reusable knowledge.

## Features

- Evidence Scout.
- Investigation Tree.
- Root Cause Debate.
- Recommendation Quality Gate.
- KPI Storyboard.
- Anonymized Pattern Library.

## Run

```powershell
python .\scripts\autonomous_intelligence.py `
  --evidence .\evidence\IT-TEST-ERP4CU `
  --output .\out\autonomous-intelligence.json
```

The dashboard embeds these results in the `Autonomous AI` tab.
